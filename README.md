# Requirements
The code needs the following packages:
- torch == 2.4.0
- torch-geometric == 2.6.1
- torch_scatter == 2.1.2
- scikit-learn == 1.5.2
- numpy == 1.26.4
- tqdm == 4.66.4



# Run
Unzip the files in the folders 'data' and 'tmp' and run:
```
python main.py
```
